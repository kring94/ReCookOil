# Título del proyecto

ReCookOil

## Descripción del proyecto

ReCookOil es un proyecto que tiene como objetivo reducir el desperdicio de aceite de cocina utilizado en hogares. La idea es permitir a los usuarios de la aplicación llevar un registro del aceite que utilizan en sus cocinas y ofrecer una manera fácil de reciclar ese aceite para su reutilización.

## Instalación

1. Clona el repositorio: `git clone https://github.com/kring94/ReCookOil.git`
2. Instala las dependencias: `npm install`


## Contribución

Si deseas contribuir al proyecto, sigue estos pasos:

1. Haz un fork del repositorio.
2. Clona tu fork: `git clone https://github.com/tu-usuario/ReCookOil.git`
3. Crea una rama para tu contribución: `git checkout -b nombre-de-tu-rama`
4. Realiza tus cambios y haz commit de tus modificaciones: `git commit -am "Breve descripción de los cambios"`
5. Realiza un push a tu rama: `git push origin nombre-de-tu-rama`
6. Crea un pull request en el repositorio original.
